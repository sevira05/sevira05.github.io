Tentang Aplikasi

Aplikasi ini memungkinkan pengguna mengambil nomor antrian secara online melalui mobile app, dan memantau status antrian secara real-time.

Admin dapat mengelola seluruh proses melalui dashboard web yang terintegrasi dengan backend API.
